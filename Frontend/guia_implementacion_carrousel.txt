GUÍA DE IMPLEMENTACIÓN: CARRUSEL DE FONDO (BACKGROUND SLIDER)
=============================================================

Esta guía resume los pasos realizados para implementar un carrusel de imágenes de fondo
que cambia automáticamente y persiste a través de la navegación de la aplicación.

PASO 1: Crear el Componente Lógico (BackgroundCarousel.jsx)
-----------------------------------------------------------
Ubicación: src/components/BackgroundCarousel.jsx

Este componente maneja el estado de qué imagen se está mostrando actualmente y
configura un intervalo para cambiarla cada 5 segundos.

Puntos clave:
- Usa `useState` para guardar el índice de la imagen actual.
- Usa `useEffect` con `setInterval` para rotar el índice.
- Renderiza todas las imágenes pero solo muestra la activa mediante CSS.
- IMPORTANTE: Las rutas de las imágenes deben ser relativas a la carpeta `public`.
  Ejemplo: Si la imagen está en `public/img/foto.jpg`, la ruta es `/img/foto.jpg`.

PASO 2: Crear los Estilos (carousel.css)
----------------------------------------
Ubicación: src/style/carousel.css

Aquí definimos que el carrusel se quede fijo en el fondo y no tape el contenido.

Claves CSS:
- .background-carousel {
    position: fixed;  /* Se queda fijo al hacer scroll */
    top: 0; left: 0;
    width: 100%; height: 100%;
    z-index: -1;      /* Se coloca DETRÁS de todo el contenido */
}
- .carousel-image {
    opacity: 0;       /* Por defecto invisibles */
    transition: opacity 1.5s; /* Transición suave (fade) */
}
- .carousel-image.active {
    opacity: 1;       /* Solo la activa es visible */
}

PASO 3: Integración Global (Layout.jsx)
---------------------------------------
Ubicación: src/components/Layout.jsx

Para que el carrusel se vea en TODAS las páginas (Login, Home, Perfil), lo
agregamos dentro del componente Layout, que actúa como contenedor principal.

Código:
  import BackgroundCarousel from "./BackgroundCarousel";
  // ...
  return (
      <div>
          <BackgroundCarousel /> {/* Se monta aquí */}
          <header>...</header>
          <main>{children}</main>
      </div>
  );

PASO 4: Limpieza de Estilos Anteriores (home.css)
-------------------------------------------------
Ubicación: src/style/home.css

Si antes tenías una imagen de fondo en el `body`, hay que quitarla para que no
interfiera o se cargue innecesariamente detrás del carrusel.

Se recomienda dejar un color de fondo sólido (ej: #1a1a1a) por si las imágenes
tardan en cargar.

RESUMEN DE ARCHIVOS NECESARIOS:
1. src/components/BackgroundCarousel.jsx
2. src/style/carousel.css
3. Imágenes en la carpeta public/img/

NOTAS ADICIONALES:
- Para agregar más imágenes, solo añádelas al array `images` en BackgroundCarousel.jsx.
- Para cambiar la velocidad, modifica el valor `5000` (ms) en el `setInterval`.
